puffkit documentation
=====================

**puffkit** is a Python library for app and game development,
providing a set of classes and functions to simplify the process of creating
applications and games.

It is designed to be easy to use, flexible and modular, allowing developers to
focus on building software without worrying about the underlying implementation.

puffkit is built on top of the amazing `pygame-ce <https://pyga.me>`_ library.
